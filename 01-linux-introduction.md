# Linux Introduction

## Overview

Linux is a free and open-source operating system family based on the Linux kernel. It manages computer hardware resources and provides an environment where users and applications can interact with the system.

Linux is widely used in servers, cloud computing, embedded systems, Android, and cybersecurity because of its stability, security, and flexibility.

# What is an Operating System?

An Operating System (OS) is system software that acts as a bridge between computer hardware and software.

Without an operating system, hardware components such as the CPU, RAM, keyboard, and storage cannot work together effectively.

### Responsibilities of an Operating System

- Manage CPU resources
- Manage memory (RAM)
- Manage files and storage
- Run applications
- Control hardware devices
- Provide an interface for users

---

# What is Linux?

Linux is an operating system family built around the Linux kernel.

It manages computer resources such as:

- CPU
- RAM
- Hard Disk / SSD
- Keyboard
- Mouse
- Display
- Network devices

Linux allows users and software to communicate with the hardware efficiently.

---

# History of Linux

- Linux was created by **Linus Torvalds** in **1991**.
- The first component developed was the **Linux Kernel**.
- Today Linux is developed by thousands of contributors around the world as an open-source project.

---

# What Does Open Source Mean?

Linux is open source, which means its source code is publicly available.

Anyone can:

- View the source code
- Study it
- Modify it
- Distribute modified versions (subject to the applicable license)

Unlike Linux, Microsoft Windows is proprietary software and its source code is not publicly available.

---

# Why is Linux Free?

Most Linux distributions can be downloaded and used without paying a license fee.

Because Linux is open source, organizations and developers can build their own Linux distributions based on the Linux kernel.

Examples include:

- Ubuntu
- Debian
- Fedora
- CentOS Stream
- Kali Linux

# Where is Linux Used?

## 1. Servers

Linux is widely used to power web servers and enterprise infrastructure.

Examples of companies that extensively use Linux in their infrastructure include:

- Google
- Amazon
- Netflix
- Meta (Facebook)

---

## 2. Cloud Computing

Linux is the most widely used operating system for cloud servers.

Examples include:

- Amazon Web Services (AWS)
- Microsoft Azure
- Google Cloud Platform (GCP)

---

- Security Analysts
- SOC Analysts
- Penetration Testers
- Ethical Hackers
- Incident Responders
- System Administrators

- Powerful command-line interface
- Excellent networking tools

## 4. Android

---

## 5. Supercomputers

# Advantages of Linux

## Free

Most Linux distributions are available without licensing costs.

## Secure

Linux has a strong permission system and is generally considered more resistant to common malware than many desktop operating systems.

## Stable

Linux systems can often run for long periods without requiring a reboot, depending on updates and maintenance.

## Fast

Linux performs well even on relatively modest hardware.

## Customizable

Users can customize nearly every part of the operating system.

# Linux vs Windows

| Linux | Windows |
|--------|----------|
| Open Source | Proprietary |
| Usually Free | Many editions require a paid license |
| Highly Customizable | Limited customization |
| Strong command-line tools | GUI-focused |
| Popular for servers and cloud | Popular for desktop users |

# Why Linux Matters in Cybersecurity

Linux is one of the most important operating systems in cybersecurity.

Many security tools are designed to work on Linux, including:

- Nmap
- Wireshark
- Burp Suite
- Metasploit
- tcpdump
- Aircrack-ng

Linux is preferred because it provides:

- Powerful networking capabilities
- Easy access to system logs
- Strong permission management
- Automation using Bash scripts
- Excellent server administration tools
